# Product Management System
This Spring Boot project handles product and category data with GST, discount, and delivery charge calculations.

## Features
- Add, update, delete, and view products
- Category-based pricing with GST, discounts, and delivery
- RESTful APIs using Spring Boot and JPA

## Tech Stack
- Java
- Spring Boot
- MySQL
- JPA / Hibernate
